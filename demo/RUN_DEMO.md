# 量潮财务工具 — 演示启动指南

## 环境要求

- Python 3.12+
- 现代浏览器（Chrome / Edge / Firefox）

## 快速启动

### macOS / Linux

```bash
chmod +x run_demo.sh
./run_demo.sh
```

### Windows

双击 `run_demo.bat`，或在终端运行：

```bat
run_demo.bat
```

启动后浏览器会自动打开 `demo/index.html`。

如果浏览器未自动打开，手动访问文件：在文件管理器中双击 `demo/index.html` 即可。

## 演示流程

启动后按页面指引操作：

```
① 录入单据 → ② 确认标准化 → ③ 系统自动分类 → ④ 批量审核 → ⑤ 统计看板
```

1. **录入**：填金额、选部门、写描述，点"录入"
2. **确认**：预览标准化结果，点"确认"
3. **分类**：系统自动按关键词预分类，无需操作
4. **审核**：在分组中点"确认以上 N 条为 XX"，一键批量通过
5. **统计**：看板自动更新汇总和图表

## 注意事项

- 演示数据为模拟数据，每次重启可重新 `seed.py` 重置
- API 服务运行在 `http://localhost:8000`
- 演示数据存储在 `demo/demo.db`（独立数据库，不碰主开发库）
- 如遇端口被占用，修改 `run_demo.sh` / `run_demo.bat` 中的端口号
